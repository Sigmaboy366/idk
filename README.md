| Key | Color  |
|----|-------|

| `2` | 🟠 Orange  |
| `3` | 🟡 Gold  |
| `4` | 🟢 Green  |
| `5` | 🔵 Blue  |
| `6` | 🟣 Purple  |
| `0` | 🔄 Reset  |
